# 标杆移植说明

仓库的工程骨架来自两类已验证范式的私域化翻译：

- 单 orchestrator、文件状态机与阶段卡点，用于团长编排和断点续跑。
- 多视角独立采样、主持人收口与二审，用于四官互不可见评审。
- 质量门：静态扫描 + 合规红线一票否决（已实装）；判官、蒙卡一致性与加权分为 4 档规划（未实装）。
- 单一入口、任务后导航、跨对话档案、知识原子和多端分发，用于把孤立 skill 组织成产品。

实现细节与逐项来源见 [`teardowns/`](teardowns/) 和 [`blueprint.md`](blueprint.md)。本项目只移植工程逻辑，不复制外部项目的业务内容。
