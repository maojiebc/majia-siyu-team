# 群发合规前置扫描

> 词库单一真源 = `src/siyu_team/eval/compliance_lexicon.py`（专家团合规官同一套，含群发裂变门槛检测）。
> 群发是企微封号重灾区，`qunfa_lint.py` 在通用红线之外**额外重点扫诱导分享**。

## 怎么用
```bash
echo "群发文案" | python3 plugins/siyu-execution/skills/siyu-qunfa/scripts/qunfa_lint.py -
```
命中即就地改写，本 skill 只输出扫描通过的文案。

## 扫描什么
- **COMPLIANCE_RED（封号/绝对化，必改）**：第一、最便宜/最好、100%、永久、稳赚…（来自共享词库）。
- **QUNFA_INDUCE（群发诱导分享，必改，封号高发）**：转发得赠、集赞、拉 X 人进群、分享到 N 个群、邀请 N 位好友才享价——这类裂变门槛是企微高危动作。
- **ABSOLUTE_CLAIM（绝对化，建议改）**：国家级、顶级、独一无二、史上最。

## 群发特有红线（除词库外的人工要点）
- **频次不刷屏**：同一群一天别超 3 条（脚本查不了频次，靠 SOP 控）。
- **优惠真实**：满减/赠品写清真实条件与有效期，不做"假大空一键转发"。
- **裂变走官方**：要裂变用企微官方「客户群/群发」和合规任务宝，不用"转发得利"诱导。

## 改写方向
- "转发到 3 个群领霸王餐" → "群里的朋友进店报暗号，直接享专属价"（去掉裂变门槛，专属靠"在群里"本身）
- "集 20 赞送菜" → "今天群内接龙前 20 名，到店送一份"（把"集赞"换成"群内接龙"，不外扩诱导）
- "全网最低价" → "群价 XX，比平台便宜 X 元"（用真实幅度）
