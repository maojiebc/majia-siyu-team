# 活动模板：偷着打折（招牌）

> 来源：`knowledge/02-industry/catering/餐饮私域方法论.md`（综合自公开餐饮私域实战）。
> 原则：**所有活动私域首发，平台联动**——不破价、养平台权重、公私域闭环。

## 玩法机制（先懂再写文案）
1. 平台后台（美团/抖音）建一个折扣力度比现有稍大的套餐（有新品/大促更好）。
2. 建完**不上架** → 联系客户经理审核通过后**隐藏**。
3. 隐藏后：普通消费者从门店主页看不到这个专属优惠。
4. 隐藏套餐二维码做成海报 + 文案，发私域（群/朋友圈/1v1）。
5. 私域好友享"专属特权"下单，走的却是平台隐藏套餐 → **每笔计入平台热度值**。

好处：① 公私域交易闭环；② 不打乱平台价格体系（不破价）；③ 养平台权重。

## 群发文案骨架（本 skill 出稿时套这个）
- **钩子（专属感）**：`【群内专属】…只有群里的人能用`
- **正文**：说清是什么套餐、优惠多少、为什么划算（对齐真实人均）
- **动作**：`扫这张图的码下单` / `进店报暗号`（走隐藏套餐二维码）
- **截止**：`只限本周` / `前 N 位`（真实名额）

示例结构（通用示意，未接真实语料）：
> 【群内专属】这个双人套餐美团上搜不到，只发咱群里 👇 原价 X、群价 Y，够两个人吃到扶墙。扫图下单，只限本周末。

## 合规红线（务必过 qunfa_lint）
- ❌ 不写"全网最低/全城第一/100%"——用真实优惠幅度表述。
- ❌ 不做"转发到 N 个群/集赞/拉人"才享价——那是诱导分享，企微封号高危。专属特权靠"在群里"本身，不靠裂变门槛。
- ✅ 优惠条件真实、有效期清晰、隐藏套餐规则合规（走平台客户经理正规审核）。
