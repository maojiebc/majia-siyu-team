# 朋友圈合规前置扫描

> 词库单一真源 = `src/siyu_team/eval/compliance_lexicon.py`（专家团合规官同一套），规则依据见 `knowledge/01-wechat-official/compliance/redlines.md`。
> 这一层就是把"合规官"从事后评审变成**生成前的内嵌函数**——边写边合规。

## 怎么用
对每条文案跑：
```bash
echo "文案" | python3 plugins/siyu-execution/skills/siyu-pyq/scripts/pyq_lint.py -
```
命中即在文案里**就地改写**再发；本 skill 只输出扫描通过的文案。

## 扫描什么（朋友圈相关子集）
- **COMPLIANCE_RED（封号/欺骗，必改）**：诱导分享、外挂群发、虚拟定位、以及广告法硬违禁词（最、第一、100%、永久、稳赚、包治…）。
- **ABSOLUTE_CLAIM（绝对化，建议改）**：国家级、顶级、绝无仅有、独一无二、史上最。

> 注：静态词库里的 `NO_METRIC / NO_CALIBRATION` 等是给"方案质量门"用的，朋友圈文案不适用，`pyq_lint.py` 已过滤，只查合规相关项。

## 餐饮特有表述边界【部分待马甲填/核对】
- **产地/工艺**：写"当天到店/现切"要能对应真实供应链，别写查不到出处的"特供/直供"。
- **功效**：食品不得宣称疗效（"养胃/降火/治…"越界）。
- **优惠**：满减/赠品要写清真实条件与有效期，不做"最低价/全网最便宜"类表述。

## 改写方向（命中后怎么救）
- "最嫩" → "涮十秒就很嫩" / "口感很嫩"
- "第一鲜" → "当天现切，图新鲜"
- "100% 好评" → "老顾客回头率很高"（或引用真实评价原文）
- "转发得福利" → 改走企微官方"客户群/群发"路径，不做诱导分享
