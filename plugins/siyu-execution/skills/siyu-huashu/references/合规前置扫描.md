# 话术合规前置扫描

> 词库单一真源 = `src/siyu_team/eval/compliance_lexicon.py`（专家团合规官同一套，含首句敏感信息检测）。

## 怎么用
```bash
echo "话术" | python3 plugins/siyu-execution/skills/siyu-huashu/scripts/huashu_lint.py -
```

## 扫描什么
- **COMPLIANCE_RED（封号/绝对化，必改）**：第一、最便宜/最好、100%、永久、稳赚…
- **QUNFA_INDUCE（诱导分享，必改）**：转发得赠、集赞、拉 X 人——欢迎语里也不能有。
- **HUASHU_PRIVACY（首句索取敏感信息，建议改）**：一上来就"留个手机号/发身份证/加我发定位"——收集个人信息要有授权口径，别在破冰第一句索取。
- **ABSOLUTE_CLAIM（绝对化，建议改）**。

## 话术特有红线（人工要点）
- **别硬推销开场**：破冰第一句是关系，不是开单（脚本查不了语气，靠 SOP 与自检）。
- **信息收集要授权**：确需手机号/会员信息，要明示用途 + 征得同意，不默认索取。
- **不承诺做不到的**：答疑别为了留客乱承诺（"随时退""绝对满意"）。

## 改写方向
- "加我先留个手机号登记" → "方便的话可以留个手机号登记会员，不留也不影响哈～"（给选择，不强制）
- "转发这条领新人礼" → "新人礼直接到店报一声就送"（去诱导分享）
- "绝对满意" → "有不合适随时跟我说，我给您处理"（去绝对化，给路径）
