# 账号 IP 立住（开户 / 改名模板）

> 来源：`knowledge/02-industry/catering/餐饮私域方法论.md`（综合自公开餐饮私域实战）。
> 破冰之前，账号本身就是第一印象——头像/名字不对，第一句欢迎语再好也白搭。

## 造 IP 公式 = 头像 + 名称 + 朋友圈背景（都关联品牌）

### 头像
- ✅ 真人 + 微笑 + 工服/后厨/门店场景
- ❌ 用 logo（除非品牌势能已经很强）——「用 logo 装狠人」是第一坑
- ❌ 网图/表情包/风景

### 名称模板
| 类型 | 模板 | 传递感觉 |
|---|---|---|
| 单店 | `门店 + 人名`（或 `+ 福利发放版`） | 老实本分手艺人、真诚 |
| 连锁 | `品牌 + 门店 + 职务 + 人名` | 干净、品质、专业、有料 |

反面（都别用）：本名「我心飞扬大哥」、「某某面馆法人代表谁谁谁」——离店后没人找得到你。

### 朋友圈背景图
= 招牌特色菜（一眼知道你是干嘛的、卖什么）。

## 改名 checklist（给已有号做体检）
- [ ] 头像是不是真人+场景？
- [ ] 名字带不带品牌/门店？换个店还认得出是谁吗？
- [ ] 背景图是不是招牌菜？
- [ ] 整体传递的是"一个人"还是"一个广告位"？

> 真实人设细节（这个人叫什么、什么调性）以 `assets/真实语料-待填.md` 为准。
