# 答疑话术库

## 应答结构（每个问题都套）
**先答要点（别绕）→ 再给一步动作 → 留人味（一句自然的）**
> 例：「几点关门？」→「今天营业到 21:30（要点）。想来的话我给您留个位（动作）。晚点人少，涮起来更从容～（人味）」

## 高频问题分类（餐饮示例，真实答案填 assets）
| 类 | 典型问题 | 应答要点 |
|---|---|---|
| 营业 | 几点开/关？周末营业吗？ | 给准确时间 + 是否需预约 |
| 位置 | 在哪？怎么走？有停车吗？ | 给定位 + 一句路线提示 + 停车信息 |
| 预订 | 能订位吗？几人桌？ | 能否订 + 怎么订（一步动作） |
| 优惠 | 有活动吗？会员怎么弄？ | 给真实优惠 + 参与方式（真实条件） |
| 配送 | 外卖吗？配送范围？ | 平台/范围/起送 |
| 退换 | 不满意怎么办？ | 给态度 + 具体处理路径 |

## 三个原则
1. **真实优先**：答案对不上真实规则就是坑，宁可说"我确认下再回您"，别编。
2. **一步就够**：每个回答只推一个下一步动作，别塞一堆链接。
3. **像人不像 FAQ**：同一意思用口语说，别复制粘贴机器腔。

## 兜底话术（答不上来时）
「这个我帮您确认一下，稍等两分钟给您准信儿～」——比乱答强，也比已读不回强。

> 真实答案（营业时间/位置/优惠/群规）填 `assets/真实语料-待填.md`；答案要与门店实际一致。
