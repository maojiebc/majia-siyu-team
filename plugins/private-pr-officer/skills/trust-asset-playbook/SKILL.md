# 信任资产打法（IP人设/朋友圈/口碑/转介绍）

## Overview
对齐方法论引擎「私域即公关」。本 skill 给方法论框架；具体话术/阈值/配比是马甲护城河，标【待马甲填真实SOP】。

## 方法框架（行业通行版，真实模板见护城河注入点）

信任是私域里少数会复利的资产。它慢，但一旦建起来，获客成本会一路往下走。

1. **IP 人设立真人，不立 logo**：私域第一印象靠头像 / 名字 / 朋友圈，真人 + 场景比冷冰冰的 logo 让人愿意留。人设要真实可持续，不是演。
2. **朋友圈养信任，不刷广告**：内容配比里价值 / 生活 / 互动占大头，硬广是少数。刷屏广告是在消耗信任，不是建立。
3. **口碑沉淀成可见资产**：好评、案例、客户反馈主动收集、公开呈现，让新客一进来就看到「别人在这得到过什么」。
4. **转介绍靠体验不靠利诱**：老客愿意转介绍，根子是服务体验值得说；机制（奖励）是催化剂不是发动机，纯利诱来的留不住。

> 具体的人设模板、朋友圈配比、转介绍机制是马甲护城河，标【待马甲填真实SOP】，由本人注入。
