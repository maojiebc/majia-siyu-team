# 私域舆情/差评应对 SOP

## Overview
对齐方法论引擎「私域即公关」。本 skill 给方法论框架；具体话术/阈值/配比是马甲护城河，标【待马甲填真实SOP】。

## 方法框架（行业通行版，真实话术/额度见护城河注入点）

差评处理是做给围观者看的——围观的人比当事人多，处理姿态就是品牌姿态。

1. **先响应速度，后解决方案**：第一时间先接住（看到了、在处理），比憋着想完美方案再回更重要。时效本身就是态度。
2. **公开台面 + 私下解决双轨**：公开评论区回一句得体、让围观者看到诚意的；具体补偿和细节拉到私信，不在公开场合讨价还价。
3. **补偿分级、不越级**：一线能拍板的额度先给，别层层上报拖成大事；超额度再升级，但升级也要快。
4. **复盘归因，不只灭火**：同类差评反复出现是产品 / 流程问题，不是话术问题，要往上游改，光靠话术兜不住。

> 各场景的真实响应话术、补偿额度、升级线是马甲护城河，标【待马甲填真实SOP】，由本人注入。
