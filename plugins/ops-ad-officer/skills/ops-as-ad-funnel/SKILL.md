# 运营即广告投放（漏斗/埋点）

## Overview
对齐方法论引擎「运营即广告」。本 skill 给方法论框架；具体话术/阈值/配比是马甲护城河，标【待马甲填真实SOP】。

## 方法框架（行业通行版，真实节奏见护城河注入点）

每个运营动作本身就是一次广告投放——自带触达、自带转化，做之前先问「这一下花的触达值不值」。

1. **把动作当投放看**：一条群发 = 一次触达采买。发之前想清楚触达谁、期待什么动作、怎么衡量，而不是「到点了该发了」。
2. **漏斗埋点对齐动作**：曝光→点击→到店 / 下单，每一环都要有可埋点的动作痕迹，否则投了不知道效果。
3. **不刷屏、不浪费触达**：同一批人一天的触达额度有限，刷屏是在提前透支后续所有触达的打开率。触达是成本，不是免费的。
4. **自带传播优先**：能设计成「用户愿意自己转」的动作（内容型 / 福利型），比硬推的动作单位触达产出高。

> 各渠道的真实频次上限、承接话术、投放节奏是马甲护城河，标【待马甲填真实SOP】，由本人注入。
