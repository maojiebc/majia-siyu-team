# 转化口径定义库（UV→加微→首单→复购）

## Overview
对齐方法论引擎「运营即广告」。本 skill 给方法论框架；具体话术/阈值/配比是马甲护城河，标【待马甲填真实SOP】。

## 方法框架（行业通行版，真实阈值见护城河注入点）

转化口径的价值不在漂亮数字，在能对账、能复盘。每一跳都定义清楚再谈优化。

1. **每跳写死分子 / 分母 / 时间窗**：加微率 = 加微数 ÷ 什么（进店 UV？扫码曝光？）÷ 什么周期。分母不同，同一个「加微率」能差好几倍。
2. **区分累计与周期**：会员数、加微数默认是累计量；转化率、活跃度是周期量。大数（如「累计 X 万会员」）一定带约束条件，别拿累计充当月活。
3. **漏斗逐跳看，不看总转化**：UV→加微→首单→复购，哪一跳掉得最狠，问题就在那跳，别笼统说「转化低」。
4. **口径先于工具**：先把口径定义写清，再谈接 BI 埋点；口径没定就埋点，埋出来的也是糊涂账。

> 各行业各阶段的真实基准值、健康区间是马甲护城河，标【待马甲填真实SOP】，由本人注入。
