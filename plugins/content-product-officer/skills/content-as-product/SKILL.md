# 把内容当产品做（选题/钩子/承接）

## Overview
对齐方法论引擎「内容即产品」。本 skill 给方法论框架；具体话术/阈值/配比是马甲护城河，标【待马甲填真实SOP】。

## 方法框架（行业通行版，真实配比/话术见护城河注入点）

内容当产品做，盯三件事——有人看、有人存、有人动。

1. **选题从处境反推，不从灵感出发**：先问这条内容替用户解决当下哪个具体处境（今天吃什么 / 怎么点不踩雷 / 家里来客怎么办），再定题。用户的真实提问、评论、客服高频问题是选题富矿。
2. **钩子放前三秒 / 首图首句**：图文看首图能不能让人停，视频看前 2 秒有没有由头。钩子不是标题党，是把「跟你有关」在第一眼说清。
3. **每条内容留一个承接动作**：看完能干什么——到店 / 进群 / 接龙 / 领券。没有承接的内容是自嗨，承接要具体到「点哪里、说什么」。
4. **跑通就沉成模板**：一条内容跑通后沉成结构（选题角度 + 钩子形态 + 承接方式），下次填变量即可，不每次从零想。

> 具体的选题库、钩子话术、内容配比是马甲护城河，标【待马甲填真实SOP】，由本人注入。
