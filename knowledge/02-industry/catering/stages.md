# 餐饮册 · 行业×阶段定义

> 团长 Step 1 路由读这里。基线数据标 【待马甲填真实数据】。

## 阶段定义
| 阶段 | 特征 | 私域重点 |
|---|---|---|
| cold 冷启动 | 0 起步 / 有微信没体系 | 加得上人 + 加进来不流失：门店钩子、承接、第一周 SOP |
| growth 扩张 | 有体系要提效 | 分层提效 + 复购：标签体系、自动化 SOP、复购召回 |
| mature 成熟 | 要规模化裂变 | 规模化裂变 + 会员体系：合规裂变、会员等级、案例复制 |

## 餐饮特有变量（团长诊断时必问）
- 门店数 / 单店日均客流 / 客单价
- 堂食 vs 外卖占比
- 复购周期（快餐 vs 正餐差异大）【待马甲填行业基线】
- 现有加微钩子（扫码点餐？会员卡？）

## 餐饮加微钩子库（冷启动承接）
> 通用脱敏钩子；真实活动力度 / 配比 / 节奏见私有 `knowledge/03-majia-sop/`。
- **进群即福利**：进群拼手气红包得代金券 / 霸王餐一份
- **到店复访钩**：加企微下次到店领免费饮料 / 小食
- **买单场景**：收银台口头引导「加店长企微领用券超优惠」（收银员当面引导的转化通常更高）
- **物料位**：店外玻璃贴 / 收银台立牌 / 门口展板，均贴店员好友码

## 冷启动第一周基建 SOP（企业微信四件套）
详见 [`../../01-wechat-official/features/企业微信基建四件套.md`](../../01-wechat-official/features/企业微信基建四件套.md)：
- **四件套按序**：好友码 → 对外资料页 → 好友欢迎语 → 群活码+进群欢迎语，缺一不成地基
- **先加好友再进群**：朋友圈 + 私聊 + 群消息三通道触达，好友码是一切源头
- **认证**：用工商字号简称或自有商标，避开他人已注册的品类词
- **老客迁移**：不一刀切解散老群，先用新群专属钩子引一波

## 方法论与招牌玩法
详见 [`餐饮私域方法论.md`](餐饮私域方法论.md)（行业参考层）：
- **私域五步骨架**：造 IP → 做链接 → 促活 → 分层 → 复购（映射冷启动/扩张/成熟三阶段）
- **造 IP 公式**：头像（真人+微笑+场景，禁 logo）+ 名称模板（单店/连锁）+ 背景图=招牌菜
- **企微地基（冷启动）**：四件套承接底座 + 企微 vs 个微取舍（见上）
- **成熟阶段招牌玩法「偷着打折」**：活动私域首发 + 平台隐藏套餐联动，不破价、养平台权重
