# 餐饮册 · 行业×阶段定义

> 团长 Step 1 路由读这里。基线数据标 【待马甲填真实数据】。

## 阶段定义
| 阶段 | 特征 | 私域重点 |
|---|---|---|
| cold 冷启动 | 0 起步 / 有微信没体系 | 加得上人 + 加进来不流失：门店钩子、承接、第一周 SOP |
| growth 扩张 | 有体系要提效 | 分层提效 + 复购：标签体系、自动化 SOP、复购召回 |
| mature 成熟 | 要规模化裂变 | 规模化裂变 + 会员体系：合规裂变、会员等级、案例复制 |

## 餐饮特有变量（团长诊断时必问）
- 门店数 / 单店日均客流 / 客单价
- 堂食 vs 外卖占比
- 复购周期（快餐 vs 正餐差异大）【待马甲填行业基线】
- 现有加微钩子（扫码点餐？会员卡？）

## 餐饮加微钩子库【待马甲填真实SOP】
- 占位示例：买单立减券需加企微领取 / 菜品抽奖 / 会员日专属

## 方法论与招牌玩法
详见 [`餐饮私域方法论.md`](餐饮私域方法论.md)（行业参考层）：
- **私域五步骨架**：造 IP → 做链接 → 促活 → 分层 → 复购（映射冷启动/扩张/成熟三阶段）
- **造 IP 公式**：头像（真人+微笑+场景，禁 logo）+ 名称模板（单店/连锁）+ 背景图=招牌菜
- **成熟阶段招牌玩法「偷着打折」**：活动私域首发 + 平台隐藏套餐联动，不破价、养平台权重
